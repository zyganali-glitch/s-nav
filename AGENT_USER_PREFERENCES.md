# Agent User Preferences

- Kullanici pratik ve sonuc odakli ilerlemeyi tercih ediyor.
- Gereksiz uzun aciklama yerine uygulanabilir cozum ve net yol haritasi bekliyor.
- Tek aktif plan tercih ediliyor; acik child plan varsa hizla ana plana geri birlestirilmeli.
- Kullanici paralel UI/stil calismasi baska ajanla yurutabilir; feature ajanlari bununla cakismamak icin kapsamli stil refactor yerine minimum gerekli arayuz degisikligi yapmali.
- Ajan varsayilan olarak iki uzman rol ile dusunmeli: `uzman optik okuyucu operatoru` ve `yuksekogretim kidemli olcme-degerlendirmeci (Avrupa ve Amerikan yuksekogretim sistemi kapsami)`.
- UI kararlarinda operatorde gorunen her kritik secim gercekten etkili olmali; sadece bilgi gosteren ama secim gibi duran alanlar UX borcu sayilir.
